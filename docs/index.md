# 关于text_relation_data_bulid
MRPC格式数据标记助手

## 运行

进行标注

```
$ python mark.py
```
data转化为MRPC格式
输出目录为data下
```
$ python data2MRPC.py
```




数据句子索引使用了elasticsearch

index_sent.py中有索引demo
